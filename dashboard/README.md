# Excel Dashboards

This folder contains final Excel dashboard outputs created for management and business reporting.

---

## Sales Overview Dashboard
**Metrics included:**
- Total Revenue
- Total Orders
- Average Order Value
- Monthly Revenue Trend

**Purpose:**  
Provides a high-level snapshot of overall business performance.

---

## Category & Regional Performance Dashboard
**Metrics included:**
- Revenue by Product Category
- Revenue by Region
- Performance comparison using bar charts

**Purpose:**  
Helps management identify top-performing categories and regions.
